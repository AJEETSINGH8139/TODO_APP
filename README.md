# TODO_APP
* This is the simple TODO app project of web development.
* In this project, we can add any task.
* We will also delete any task if we want.
* There should also be a button with the task: by which we can know about the task.

# Tools
* In this project, I used JavaScript, ejs, CSS.
* I used Node.js for runtime environment for developing server-side and networking applications.
* I used MongoDB for database.
* Express is also used.
